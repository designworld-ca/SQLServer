Verifies CIS control 16.9 Configure Account Access Centrally


1. logon as administrator
2. Click Start
3. Select Run 
4. enter gpedit.msc 
5. Click OK to open Local Group Policy Editor.
6. Expand Computer Configuration -> Windows Settings -> Security Settings -> Account Policies -> Password Policy
7. Take a screenshot of the settings
8. Take a screenshot which shows whether you can change these settings
